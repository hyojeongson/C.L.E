Minimal CSS Framework

MinimalCSS is a free CSS template released under the MIT License. The MIT License is simple and easy to understand and it places almost no restrictions on what you can do with this template. You are free to use any MinimalCSS in any project (even commercial projects) as long as the copyright header is left intact.

MinimalCSS was designed for simple static websites, but can also be adapted to fit a CMS of your choice. It does not rely on any outside libraries or services and is suitable for use on intranets.

After downloading, simply edit the html with your content and upload it to your website. Layout examples are included in the CSS Grid section seen below.

Simple

MinimalCSS is as simple as possible and only contains two files needed for production. index.html & minimal.css

Compact

The CSS file or Cascading Style Sheet included is a compact as possible, feel free to modify to suit your needs.

Cross Browser

MinimalCSS is tested in the latest versions of Safari, Firefox, Chrome, Opera and Internet Explorer.
